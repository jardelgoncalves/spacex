## Install dependencies
```bash
npm install
# or
yarn
```

## Run the development server
```bash
npm run dev
# or
yarn dev
```

## Run the production server
First, run build command

```bash
npm run build
# or
yarn build
```
<br />

Run production mode

```bash
npm run start
# or
yarn start
```
